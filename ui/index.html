Quick start

  1. Upload a PDF, DOCX, TXT, or Markdown file in the left panel.
  2. Wait for ingestion to finish (first run downloads the embedding model).
  3. Type a specific question and click Run query.
  4. Review the answer and source chunks with relevance scores.

Set GROQ_API_KEY in .env on the API server.
